# 🧮 C# 電卓アプリ（学習用）

## 📌 概要
C# の学習目的で作成したコンソール電卓アプリです。  
入門書で学んだ基礎知識（変数・if文・クラス）を実際のアプリとして実装しました。

## 🎯 作った目的
- C# の基礎文法を実際に動くアプリで体験する
- クラスの役割分担（責務の分離）を理解する
- GitHub でのバージョン管理を習慣にする

## ✨ 機能一覧
- 四則演算（足し算・引き算・掛け算・割り算）
- ループ処理（何度でも連続して計算できる）
- エラーハンドリング（数字以外を入力してもクラッシュしない）
- 計算履歴の表示（`h` キーで今までの計算を一覧表示）

## 🛠️ 使用技術
- 言語：C# (.NET 10)
- 実行環境：コンソールアプリ（macOS）
- バージョン管理：Git / GitHub

## 📚 学習したこと
| 項目 | 内容 |
|------|------|
| 変数・型 | `string` `double` `int` の使い分け |
| 条件分岐 | `switch / case` で演算子を判定 |
| ループ | `while(true)` + `break / continue` |
| クラス | 役割ごとにクラスを分ける設計 |
| エラー処理 | `double.TryParse` で安全な入力処理 |
| コレクション | `List<string>` + `foreach` で履歴管理 |

## 🚀 実行方法
```bash
# リポジトリをクローン
git clone https://github.com/ando625/C--Calculator.git

# フォルダに移動
cd C--Calculator

# 実行
dotnet run
```
